Get an instance of RetryInterceptor that uses a default full jitter backoffretry strategy.

The retry strategy by default retries maximum 3 times, uses 800 milliseconds asthe default base delay and uses 8 seconds as default maximum backoff delay before a retry.

Return

The retry interceptor.
