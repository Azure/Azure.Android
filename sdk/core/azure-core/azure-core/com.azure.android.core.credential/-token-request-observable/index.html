TokenRequestObservable

public final class TokenRequestObservable

An Observable to send the token request from a background thread to an Observer in the main UI thread.

A use case of this type is - sending a token request from androidx.lifecycle.ViewModel code running in a background thread to the main UI thread, where the actual acquisition of token is interactive, e.g. it involves android.app.Activity UI. This Observable decouples the androidx.lifecycle.ViewModel from the UI so that the former will not hold a reference to the UI component, holding such reference can lead to a memory leak. However, the Observer of this Observable can hold a UI reference, because the Observer is removed when the UI that the Observer is associated with is in the DESTROYEDstate.
