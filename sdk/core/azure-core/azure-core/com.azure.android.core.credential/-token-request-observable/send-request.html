sendRequest

TokenRequestHandle sendRequest(List<String> scopes)
Content copied to clipboard

Emits a token request to a TokenRequestObserver registered through observe.

The Observer's onTokenRequest will be executed in the main UI thread.

Return

The handle that the caller can use to synchronously receive an access tokenproduced by the main UI thread.

Parameters

scopes

The scopes required for the token.

Throws

java.lang.IllegalStateException

if this method is called from the main UI thread.
