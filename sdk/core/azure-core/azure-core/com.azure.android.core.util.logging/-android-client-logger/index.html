AndroidClientLogger

final class AndroidClientLogger implements ClientLogger

This is a fluent logger helper class that implements the ClientLogger interface and uses the Android Log class and its methods.

The default log level is INFO

Log level hierarchy

  1. Error
  2. Warning
  3. Info
  4. Verbose
